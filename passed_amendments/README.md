# Naming Convention

The filename of passed constitutional amendments from each edition should follow the following naming convention.
```
<date-of-passing(YYYYMMDD)>-<Host IIT tag>-amendments.pdf
```
# Document Content Requirements

When drafting the final passed amendments document, it is mandatory to include the following:
* **Convener Details:** The name(s) of the Convener(s) presiding over that edition.
* **Rationales:** A clear, written rationale accompanying each passed amendment to provide context and historical reasoning for future organizing teams.
